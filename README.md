# 👤 Personal Profile Webpage

A simple and responsive personal profile webpage built using **HTML** and **CSS**. This project displays personal information, skills, hobbies, and contact details in a clean card-based layout.

## 📌 Features

- Responsive design
- Circular profile picture
- Card-based layout
- Personal information section
- Skills & hobbies section
- Contact information
- Clean and modern UI
- Built using only HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Profile-Website/
│── profile.html
│── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Vishesh786900/Profile-Website.git
```

2. Open the project folder.

3. Run the project by opening `profile.html` in any web browser.
