# 📝 Student Registration Form

A simple and responsive **Student Registration Form** created using **HTML5**. This project collects basic student information such as personal details, academic information, and social profile links.

## 📌 Features

- Student Name
- Email Address
- Phone Number
- Roll Number
- Gender Selection
- Branch Information
- Academic Year
- Date of Birth
- GitHub Profile Link
- LinkedIn Profile Link
- Submit and Reset Buttons
- HTML5 Form Validation using `required` attributes

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
Student-Registration-Form/
│── index.html
│── README.md
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Vishesh786900/Student-Registration-Form.git
```

### 2. Open the Project

Navigate to the project folder.

### 3. Run the Project

Open the `index.html` file in any modern web browser.

# 👤 Personal Profile Webpage

A simple and responsive personal profile webpage built using **HTML** and **CSS**. This project displays personal information, skills, hobbies, and contact details in a clean card-based layout.

## 📌 Features

- Responsive design
- Circular profile picture
- Card-based layout
- Personal information section
- Skills & hobbies section
- Contact information
- Clean and modern UI
- Built using only HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Profile-Website/
│── profile.html
│── README.md
```
